Για την δημιουργία του εκτελέσιμου για το B+ δέντρο, τρέξτε την εντολή:
make bp;

Για να τρέξετε το εκτελέσιμο:

./build/bp_main


- BPLUS_DATA_NODE: στο struct αυτο αποθηκευονται τα μεταδεδομενα καθε κομβου δεδομενων(αριθμος εγγραφων κλπ) στην αρχη ενος block δεδομενων
    βοηθητικές σαρτήσεις:
        - insert_rec_in_datanode: προσθετει μια εγγραφη στο block με id node που δινεται ως ορισμα. Εαν υπαρχει χωρος στο block, η εγγραφη προστιθεται
        στο block αυτο. Εαν το block δεν εχει καμια εγγραφη, εισαγεται απλα αμεσως μετα τα μεταδεδομενα, αλλιως εντοπιζεται η πρωτη εγγραφη με μεγαλυτερο 
        κλειδι απο τη νεα και εισαγεται πριν απο αυτη. Προκειμενου ολες οι επομενες εγγραφες να μετακινηθουν μια θεση δεξια, χρησιμοποιειται η memmove 
        και ετσι η νεα εγγραφη εισαγεται στην καταλληληλη θεση. Εαν δεν υπαρχει χωρος στο block, τοτε οι εγγραφες μαζι με τη νεα πρεπει να ισομοιραστουν σε 
        δυο block. Την περιπτωση αυτη την υλοποιει η συναρτηση split_data_node.
